# Imagens para Visão Executiva

Coloque as 3 fotos nesta pasta com estes nomes exatos:

1. **client-po.webp** - Foto representando Cliente/Product Owner
2. **dev-team.webp** - Foto representando Time de Desenvolvimento
3. **ceo.webp** - Foto representando CEO/Executivo

## Formato recomendado:
- Tipo: **WebP** (melhor compressão que JPG/PNG)
- Tamanho: 400x400px (quadrado)
- Peso: < 100KB cada (WebP é muito eficiente!)

## Por que WebP?
- Arquivos 25-35% menores que JPG
- Qualidade visual igual ou superior
- Suportado por todos navegadores modernos
- Carregamento mais rápido

## Como converter para WebP:
- Online: https://squoosh.app/
- Photoshop: Salvar como > WebP
- Ferramentas: cwebp, ImageMagick

## Como funciona:
As imagens na pasta `public/` são servidas diretamente pelo servidor.
O caminho `/images/client-po.webp` aponta para `public/images/client-po.webp`

Não precisa "importar" nada - só colocar os arquivos aqui!
